# 절대 길이 단위

  • px : 픽셀(화소)
  • mm : 밀리미터
  • cm : 센티미터
  • in : 인치
100px
100mm
10cm
7inch

# 상대 길이 단위

  • % : 부모 요소의 일정 비율을 차지하는 단위
  • em : 해당 요소에 설정된 font-size에 비례한 크기 설정 단위
  • rem : 현재 웹 페이지(html)에 설정된 font-size에 비례한 크기 설정 단위
  • vw : viewport width에 비례한 크기 설정 1vw는 화면 너비의 1% 크기를 의미한다
  • vh : viewport height에 비례한 크기 설정 1vh는 화면 높이의 1% 크기를 의미한다
50%(부모 요소인 body의 50%를 차지)
10em(10글자 들어갈 정도)
10rem(10글자 정도, html 태그 비례)
25vw(viewport 너비의 1/4)
25vh(viewport 높이의 1/4)
