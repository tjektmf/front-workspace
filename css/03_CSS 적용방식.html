# 웹 페이지에 CSS를 적용하는 방식 3가지

1. inline CSS (줄 내부 CSS)
- style attr을 이용해 해당 요소에 스타일을 부여하는 방식
- 적용 우선순위가 가장 높음
- style 속성의 값으로 CSS 문법을 넣을 수 있음
2. internal CSS (페이지 내부 CSS)
- <style> 태그를 이용해 현재 웹 페이지에 스타일을 부여하는 방식
- 더 나중에 나온 스타일 시트가 앞의 스타일 시트를 덮어씀
3. external CSS (페이지 외부 CSS)
- 외부의 .css 파일을 불러와 사용하는 방식
- <link> 태그를 사용해 외부의 css파일을 불러와 사용한다
- 더 나중에 나온 스타일 시트가 앞의 스타일 시트를 덮어씀
