# 반응형 웹 사이트

  • viewport의 크기에 따라 요소의 모양이 달라지는 웹 사이트
  • 각 viewport 크기에 알맞는 화면을 제공하기 위해 사용됨
  • CSS의 media query라는 문법을 사용한다
  • 기본 문법 : @media (미디어종류) and (원하는크기)

# 미디어 종류

  • all : 모든 미디어
  • print : 프린터 출력 화면의 크기
  • screen : 컴퓨터, 태블릿, 스마트폰 등의 일반 기기
  • speech : 스크린 리더(화면 읽어주는 프로그램) 등
