# CSS (Cascading Style Sheet)

  • CSS는 해당 웹 페이지의 스타일(디자인)을 담당한다
  • 웹 브라우저는 정의된 CSS를 보고 해당 요소를 어떻게 화면에 그려야 할지를 결정한다
  • CSS 스타일시트를 통해 요소의 디자인을 원하는대로 수정할 수 있으므로 HTML 태그의 기본 디자인은 거의 쓸모가 없다
  • 결국 HTML은 해당 태그가 가지고 있는 의미가 가장 중요함 (뼈대 역할)
