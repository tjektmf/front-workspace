# margin

  • 해당 요소와 다른 요소의 거리를 결정하는 여백
  • border 바깥의 여백
  • margin-bottom과 margin-top이 만나는 곳에서는 둘 중 더 높은 margin으로 적용된다
  • margin을 auto로 설정하면 해당 요소를 가운데로 정렬해준다

# padding

  • 해당 요소의 border 안쪽부터 컨텐츠까지의 여백
  • 기본적으로는 padding의 크기가 요소 전체 크기에 가산된다
    요소의 크기는 width + padding + (border-width) x2

# HTML Box Model

  • HTML의 모든 요소는 박스 형태로 구성되어 있다
  • 요소를 구성하는 박스의 제일 바깥쪽 여백이 margin이고 border, padding, content-box 순으로 요소를 구성하고 있다
  • 해당 요소의 여백을 F12 개발자 도구를 통해 그림으로 확인할 수 있다
  • 전체 요소의 크기를 설정하기 매우 어렵다

# box-sizing property

box-sizing : content-box (default)
width/height property가 content-box의 크기를 의미하게 된다
box-sizing : border-box
width/height property가 해당 요소의 전체 크기를 의미하게 된다
