# Pseudo element

  • 해당 요소의 특정 부분을 선택하는 선택자
  • 선택자 뒤에 ::을 붙여 원하는 부분만 선택한다
  • ::first-letter - 첫 번째 글자
  • ::first-line - 첫 번째 줄
  • ::selection - 블록 지정된 부분
  • ::before - 해당 요소의 맨 앞 부분
  • ::after - 해당 요소의 맨 뒷 부분
