# Position property

  • 포지션을 설정한 요소는 포지션이 설정되지 않은 요소들을 무시하면서 자유롭게 요소를 배치할 수 있다
  • static(default) - 포지션이 설정되지 않은 상태
  • absolute - 절대적인 위치를 설정
  • relative - 상대적인 위치를 설정
  • fixed - 화면에 고정할 위치를 설정, 화면 스크롤을 움직여도 고정된 자리에 위치함
  • sticky - 스크롤이 변하면서 고정될 위치를 설정
  • 포지션이 겹치는 요소가 있다면 z-index 속성으로 어떤 요소가 위에 있을지를 설정할 수 있다(z축 설정 기능)
  • 기본 z-index는 0이고 기본 요소보다 밑에 깔리도록 설정하고 싶다면 z-index를 음수로 설정할 수도 있다

# position: absolute

  • 부모 요소 내부에서 절대적인 위치를 설정
  • 만약 부모 요소에 포지션이 설정되어 있지 않다면(=static) 더 위에 있는 조상으로 거슬러 올라감
  • 포지션을 설정하고 나면 top, left, bottom, right를 통해 위치를 수정할 수 있다
