# Float

  • 해당 요소를 float으로 설정하면 다른 내용들이 해당 요소를 비켜가게 만든다
  • float : left로 설정하면 해당 요소가 왼쪽에 떠있게 되고 float : right로 설정하면 오른쪽에 떠있게 된다
  • 한글에서 '일반글자처럼 취급' 이랑 느낌이 비슷하다고 보면 될 듯
  • float을 같은 방향으로 설정한 요소는 차곡차곡 쌓인다, 오른쪽에 두개 놓았으면 먼저 놓은놈이 맨오른쪽임
  • float을 다른 방향으로 설정한 요소는 각자의 방향으로 쌓인다
  • clear 속성을 통해 해당 줄에 요소가 계속 쌓이는 것을 막을 수 있다
  • float이 설정된 요소는 해당 요소를 포함한 부모 요소의 크기 결정에 영향을 미치지 못함
  • float요소가 커진다고 이 요소를 담고 있는 부모 요소의 크기가 같이 커지지 않고 튀어나옴
