# a(anchor) tag

- a 태그는 전체 글의 일부에 링크를 걸 수 있는 태그임
- 자신의 영역을 생성하는 태그가 아니다
- href 속성에 가고 싶은 곳의 주소(url)를 설정해 놓는다
- 절대 경로와 상대 경로를 모두 사용할 수 있다

우리 깃허브로 이동

네이버로 이동

http:가 없는 네이버로 이동

http://가 없는 네이버로 이동

# URL 경로의 종류

naver.com : 상대경로 (현재위치)

./naver.com : 위와 같음 (./는 생략가능)

/naver.com : 도메인부터 시작되는 상대경로

//naver.com : 프로토콜부터 시작되는 상대경로

http://naver.com : url의 절대경로
