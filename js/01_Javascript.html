# Javascript

  • 웹 점유율이 100%인 프로그래밍 언어
  • 웹 브라우저가 해석하여 실행할 수 있는 프로그래밍 언어
  • 최근에는 해당 언어를 해석하는 성능이 좋아져 다른 용도로도 많이 사용된다 (node.js를 통해 서버 구축 등)

# 웹 페이지가 해석되는 순서

  1. 브라우저가 웹 전체 코드를 통해 HTML 구조를 먼저 계층적으로 정리함 (어떤 요소 내부에 어떤 요소가 있는지 등)
  2. 화면에 그릴 순서를 결정
  3. 각 요소의 CSS 스타일을 취합하여 결정한다
  4. 레이아웃을 설정하고 화면을 그린다
  5. 4번 과정 도중 자바스크립트를 만나면 해당 과정을 중단하고 자바스크립트를 실행한다

# 자바스크립트로 할 수 있는 일

  • 요소의 스타일 변경
  • 요소에 이벤트 추가
  • 웹 페이지에 요소의 추가 수정 삭제
