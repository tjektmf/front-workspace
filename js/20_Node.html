# Node

  • document 내부에 있는 모든 것을 Node라고 부름
  • Element보다 더 작은 개념
  • 하나의 요소를 Element Node와 Text Node로 나눌 수 있다
  • 주석들은 Comment Node라고 부름
  • JS에서 새 노드 객체를 생성하여 기존 노드에 추가할 수 있다

# 자식 요소 추가하는 메서드

  • element.appendChild() : 맨 뒤에 새 노드를 추가
  • element.insertBefore(newNode, 기준점 노드)
  • element.removeChild(node)
  • element.replaceChild(newNode, oldNode)

# 연습문제

      버튼을 누르면 out에 별이 추가되거나 제거됨
      모든 버튼이 작동
      각 별의 오른쪽 맨 위 모서리에는 해당 별이 몇번째로 추가되었는지 알 수 있는 번호가 있어야 함
