# Document Object Model

  • 웹 브라우저는 HTML 코드를 해석하여 구조화된 객체 형태로 만들어 놓는다
  • 미리 만들어져있는 document라는 객체를 통해 해당 HTML 요소에 접근할 수 있다

# Document에서 원하는 요소를 선택할 수 있는 메서드

  • document.getElementById('id')
  • document.getElementsByTagName('tagname')
  • document.getElementsByClassName('classname')
  • document.getElementasByClassName('id')
  • document.querySelector('cssSelector')
  • document.querySelectorAll('cssSelector')

# 선택한 HTML 요소의 속성 및 메서드

  • element.innerText : 해당 요소의 내용
  • element.속성이름 : 해당 요소의 해당 속성에 접근
  • element.setAttribute() : 해당 요소의 속성을 메서드로 변경
  • element.addEventListener() : 해당 요소에 이벤트 추가
  • element.style.property : 해당 요소의 스타일에 접근
  • element.className : 해당 요소의 클래스 속성에 접근
