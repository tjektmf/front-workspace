# 산술 연산자

  • + - * / %
  • 자바와 다르게 /가 몫만 구하는 것이 아니라 정확하게 계산을 함
  • 몫만 구하고 싶을 때는 parseInt()로 소숫점 아래를 없애버림

# 비교 연산자

  • <, >, ≤, ≥
  • ==, != : 값이 일치(불일치)하면 true, 타입을 신경 쓰지 않음
  • ===, !== : 값과 타입이 일치(불일치)하면 true

# 단항 연산자

  • ++
  • --
  • ! : true를 false로, false를 true로
  • - : 양수를 음수로, 음수를 양수로

#논리 연산자

  • &&
  • ||

# 할당(복합 대입) 연산자

  • +=
  • -=
  • *=
  • /=
  • %=
  • &=
  • |=

# 삼항 연산자

비교 ? true인 경우 : false인 경우;
